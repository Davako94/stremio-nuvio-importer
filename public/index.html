⚠️ Server non ancora configurato

Per abilitare il push cloud devi impostare due variabili d'ambiente su Render:

SUPABASE_URL e SUPABASE_ANON_KEY

Come ottenerli (1 volta sola):
1. Installa HTTP Toolkit sul PC
2. Collega il telefono e intercetta il traffico di Nuvio
3. Accedi a Nuvio → cerca richieste verso *.supabase.co
4. Copia l'URL e l'header apikey
5. Impostali come env vars su Render e riavvia

✅ Backup pronto!

  1. Opzione A (consigliata): Usa il Push al Cloud qui sopra — riattiva poi il Nuvio Sync
  2. Opzione B (manuale): Scarica il file → Nuvio → Impostazioni → Backup → Ripristina → poi fai subito Push to Cloud dall'app

📖 Come ottenere i backup

  1. Backup Stremio: Usa Sidekick (Libreria → Esporta) o Stremio Manager
  2. Backup NUVIO: NUVIO → Impostazioni → Backup → Crea backup
  3. Per il Push Cloud: accedi con le stesse credenziali del tuo account Nuvio

🔒 Credenziali usate solo per autenticarti con Supabase • Mai salvate sul server
